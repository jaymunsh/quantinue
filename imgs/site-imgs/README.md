# Quantinue Q 아이콘 자산

`quantinue-q-minimal.png` 콘셉트를 작은 화면에서도 선명하게 보이도록 다시 그린 실사용 세트입니다.

## 권장 파일

- 웹 로고 및 밝은 배경: `svg/quantinue-q-master.svg`
- 기본 앱 아이콘: `svg/quantinue-q-blue-bg.svg`
- 다크 테마 앱 아이콘: `svg/quantinue-q-dark.svg`
- 인쇄 및 단색 문서: `svg/quantinue-q-monochrome.svg`
- 브라우저 파비콘: `favicon/favicon.ico` 또는 크기별 PNG
- 모바일·PWA: `app-icons/`의 180, 192, 512, 1024 PNG

## 컬러

- Quantinue Blue: `#0B4DDB`
- Midnight: `#07152F`
- Electric Blue: `#2F7BFF`
- Warm White: `#FFFDF5`
- Monochrome Ink: `#111827`

## 사용 원칙

- 로고 둘레에는 Q 외곽 두께의 절반 이상 여백을 유지합니다.
- 32px 이하에서는 워드마크 없이 Q 심볼만 사용합니다.
- 밝은 배경에는 파란 Q, 파란 배경에는 흰 Q를 사용합니다.
- 임의로 늘이거나 기울이거나 색상 그라데이션을 추가하지 않습니다.

## HTML 예시

```html
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" type="image/svg+xml" href="/quantinue-q-master.svg">
<link rel="apple-touch-icon" href="/apple-touch-icon-180.png">
```
